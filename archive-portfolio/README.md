# Archived Portfolio (Next.js)

This folder contains the previous **Next.js 14** portfolio that was live before the RohanVerse React rebuild.

It is kept for reference only. The active site is built from the repository root with **React + Vite**.

## Previous stack

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion

## Run locally (archived version)

```bash
cd archive-portfolio
npm install
npm run dev
```
